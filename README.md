# ecomerce-Laravel-11
Ecomerce using laravel
 
# installation
```
composer install --ignore-platform-reqs

```
php artisan key:generate

```
cp .env.example .env

```

```
php artisan config:clear
php artisan cache:clear

```

# run
```
npm run dev
```

```
php artisan serve
```
